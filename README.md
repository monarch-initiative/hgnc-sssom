### HGNC SSSOM

Prototype script for generating HGNC SSSOM file(s)


### Installing

Install poetry - https://python-poetry.org/docs/#installation

Then run make:

```make```

### Downloading data file(s)

```
make download
```

### Running
```poetry run ./hgnc.py hgnc.yaml --output hgnc_uniprot.tsv```
